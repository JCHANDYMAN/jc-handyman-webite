Future Upgrade Rule

From now on, the easiest and safest upgrade style is:

  • Upload new root pages beside app.html.
  • Run one migration SQL file in Supabase.
  • Add one link in the CRM only when needed.
  • Do not replace the whole CRM unless it is a major stable foundation release.

Install Checklist

1. Run migration SQL 2. Upload root HTML files 3. Commit changes 4. Wait for Pages deploy 5. Test direct URL
