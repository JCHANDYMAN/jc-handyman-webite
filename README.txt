JC Handyman Cloud CRM V16 Stable Foundation

This version is a new clean foundation built from the stable CRM features.

New:
- Integrated Upgrade/System Manager inside the main CRM
- Health check
- Upgrade history
- Customer Hub
- Business Dashboard 2.0
- Work orders
- Inventory/materials
- Route planner
- Keeps customers, jobs, estimates, invoices, receipts, inspections, uploads, signatures, and paperwork

Install:
1. Run SUPABASE_SETUP.sql in Supabase SQL Editor.
2. Open config.js and replace YOUR_SUPABASE_PUBLISHABLE_KEY.
3. Upload all files to GitHub and commit.
4. Open app.html and press Ctrl + Shift + R.
5. Click System Manager and run Health Check.
