Install:
1. Run SUPABASE_SETUP.sql in Supabase SQL Editor.
2. Open config.js and replace YOUR_SUPABASE_PUBLISHABLE_KEY.
3. Upload all files to GitHub and commit.
4. Open app.html and press Ctrl + Shift + R.
